## 🛠 How To Use

### 1️⃣ Add New Blog Card

Copy this inside any section:

```html
<div class="blog-card">
    <img src="images/yourimage.jpg" alt="Blog Image">
    <div class="blog-content">
        <h3>Your Title</h3>
        <p>Your short preview text here.</p>
        <a href="#" class="read-more">Read More</a>
    </div>
</div>

Commit changes → Your site updates automatically.

🚀 Deployment (GitHub Pages)

Go to Repository Settings

Click Pages

Select branch: main

Save

Your site will be published in 1–2 minutes.
